## Open Core

- CLI
- schemas
- local protocol
- templates
- examples
- compatibility assets
- documentation

## Commercial Layers

- premium templates;
- guided setup kits;
- hosted dashboard;
- scheduled cloud runs;
- team analytics;
- enterprise runners;
- consulting and training.

## Roadmap

### 0.1

Local CLI, schemas, reports, safety defaults, docs-first site.

### 0.2

Playwright adapter, runnable demo, JSON reports, regression fixtures.

### 0.3

Chrome DevTools MCP adapter, triage workflow, richer evidence capture.

### 1.0

Stable schema contracts, adapter API, install scripts, release automation, and
commercial licensing path.
