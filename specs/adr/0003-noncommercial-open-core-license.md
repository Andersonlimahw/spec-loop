# ADR-0003 — Noncommercial Open-Core License

## Status

Accepted

## Context

The project should accept contributions and allow broad learning/internal use,
but only Anderson Lima may monetize SpecLoop commercially unless a separate
commercial license is granted.

Traditional permissive licenses do not enforce that constraint.

## Decision

Use PolyForm Noncommercial 1.0.0 for the public core and document a future
dual-license path for commercial use.

## Consequences

- Good: matches the stated monetization constraint.
- Good: keeps contributions possible with clear rules.
- Bad: not OSI-open-source by strict definition.
- Risk: some companies avoid noncommercial licenses; offer a clean commercial route.
