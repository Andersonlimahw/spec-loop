# Monetization

SpecLoop starts as a source-available open-core project.

## Open Core

- CLI
- schemas
- local protocol
- templates
- compatibility agents and skills
- examples
- docs

## Commercial Layers

- premium templates;
- guided setup kits;
- hosted dashboard;
- scheduled runs;
- team analytics;
- enterprise/self-hosted runners;
- services and training.

Only Anderson Lima may commercialize SpecLoop unless a separate commercial
license is granted.
