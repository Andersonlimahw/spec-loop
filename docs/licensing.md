# Licensing Strategy

SpecLoop uses PolyForm Noncommercial 1.0.0 for the public core.

This matches the project constraint:

- allow personal, educational, research, nonprofit, and internal noncommercial use;
- accept contributions;
- prevent third parties from reselling or operating SpecLoop commercially without permission;
- preserve Anderson Lima's exclusive commercial monetization rights.

This is source-available, not OSI-open-source. Commercial users should request a
separate license.
